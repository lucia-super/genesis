# genesis 创世纪

## 项目目标

根据模型化开发设计理念，快速构建一个新的业务系统。

1. 配置一个模块名称，快速生成与此模型相关的数据模型，列表文件，增删改查方式
2. 模型化开发，统一设置框架数据，比如样式，权限等，可以全局定制
3. 可配置管理
4. 定制化开发，如果需要不同的业务，在生成的文件结构上去修改增加业务

适合各类管理系统。
